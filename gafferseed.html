GafferSeed

GafferSeed (unofficial name) is a set of C++ libraries and python modules that integrate appleseed into Image Engine's Gaffer lighting and look development application and Cortex VFX libraries.

The code is merged into ImageEngine's official repositories.
appleseed and GafferSeed will be included in future official Gaffer releases.

Features:

  • Interactive and batch rendering
  • Camera, transformation and deformation motion blur
  • OSL shader networks
  • Automatic instancing (optional)
  • appleseed support for Cortex VFX libraries both in C++ and Python
