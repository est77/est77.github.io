Pipeline and miscellaneous tools

A small selection of pipeline tools and other projects

Alembic exporter for Blender

Developed in 2011. C++ and Python. Used in production.

Features:

  • Object hierarchies
  • Custom properties saved as uniform geoParams or user data
  • Polygon meshes with modifiers, uvs, vertex colors and weights
  • Subdivision surfaces with edge creases
  • Material indices and facesets
  • Fluid meshes with per vertex velocity
  • Frame relative sampling

Automatic Maya shading nodes generator

A python script that used metadata stored in compiled RSL shaders to create Maya python plugins and attribute editor templates representing the shaders.
It allowed artists to use custom RSL shaders in Hypershade networks as if they were native Maya nodes.

Shader unit testing

An unit testing system for shader development, based on sample scenes and reference images, written in Python.
